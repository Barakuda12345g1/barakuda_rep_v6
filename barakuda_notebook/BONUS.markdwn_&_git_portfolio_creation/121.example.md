# <center>Программный код и цитирование</center>

Для выделениея обратного программног кода используется обратный апостроф.

  * Один апостроф - для выделения кода
  
        `print('Go')`
  * Два апострофа - для выделения кода где есть апостроф
  * Три апострофа - для выделения блока кода
  ```python
        def example_func(arg):
            a = 5
            b = 6
            c = a + b

            return c
  ```

# <center>Формулы</center>
* Когда мы стафим один знак $ текст станофиться формулой

  $a = b + c$
* Два знака $ - автоматически центрируют текст

  $$a = 0$$




* Для формул иполбзуется - \

  `$\alpha$` = $\alpha$

  `$\gamma$` = $\gamma$

  `$\sigma$` = $\sigma$

* Для степеней и индекстов используем - ^ и _

   `$a^2$` = $a^2$

   `$b_{1j}$` = $b_{ij}$

   `$w^{ij}_n$` = $w^{ij}_n$

* Для создания дробей используется оператор \frac

  `$$\frac{1 + x}{n}$$` = $$\frac{1 + x}{n}$$

*Ум* заостряется от **книг** словно ***меч***
